Mac10.10.3+Hexo+Git部署静态博客

发表于 2015-08-19

Ice Break

从2009年接触博客，到后来的一度中断，到现在的想记录下来一部分属于自己的文字。
由于静态博客的流行，所以想到了最近大热的Hexo，让人兴奋的是，git可以免费托管访问。这是让我最为欣赏的。
本来还想着要不要搞个个人域名，好歹做做样子，不过最终看到个人域名的价格，还是作罢。继续支持Git！

Start

第一篇就用来记录Mac上如何部署Hexo，以及如何托管到Git上的。
人老了，记不住哇，所以还是写下来比较好一些。。。

Procedure

Mac

环境：RMBP 10.10.3， Homebrew

Homebrew相信Mac下的小伙伴都不舍得错过，这里不多讲，安装也非常简单。

Node

Hexo是基于Node的，所以要将Node安装上啦。

Hexo

Git

这里git通过brew安装

安装完了git，要将SSH Key进行和github的绑定。

  • 检查SSH Key
  • 备份已有Key(没有可以忽略)
  • 生成新Key
  • 将新Key添加到github上

登录Github，Settings -> SSH keys -> Add SSH key

将你的新生成的id_rsa.pub文件中的key复制过来。保存

  • 测试Key的有效性

如果出现了一些连接成功，并且说github不支持ssh的方式，那说明Ok了

  • 设置个人访问信息

  • 登录github，新建名为xxxx.github.io的repository,待会修改hexo的git推送配置的时候使用

  • 修改hexo的git配置，方便将hexo的内容推送到git上，由git papa托管啦就。修改hexo_blog/_config.yml

Send To Git
